# Prompt format

Preferred deterministic format:

```text
SLIDE 1 Title: ... • bullet ... • bullet ...
⸻
SLIDE 2 Title: ... • bullet ...
...
```

Supported alternatives:

- `SLIDE N ...` with line breaks
- numbered lines (`1) ...`, `2) ...`) for quick multi-slide drafts

Tips:

- Use `Title:` for clear extraction.
- Use `•` for bullet boundaries.
- Keep one major idea per slide.
